Movie Recommendation System using TMDB API

Showcase details on the movie selected and recommend similar movies.

This application provides options for users to create a recipe, save ingredients from recipes to shopping list, edit a recipe and add ingredients for recipes the user wishes for.

Created class repositories for JPA implementation. Used Hibernate to query the database and perform other CRUD operations. Postman to test the crud operations like get, post, put and delete operations.

Additional feature where registered users can chat with other users to get information on recipes and ingredients.

Technologies:
  • - Java
  • - Spring Rest API
  • - Angular 8
  • - MySQL WorkBench
  • - Hibernate
  • - HTML & CSS
  • - TypeScript
  • - Postman

I am still working on this project by adding more features..I would consider this to be one of my most memorable projects in my computer science journey.
